Installing our Chrome extension is as easy as 1, 2, 3!

1. Download and unzip

Download here. Extract the downloaded file ("hq-dataprofiler-chrome-extension.zip").


2. Add to Chrome

In a new Chrome tab, visit the link: chrome://extensions. Enable developer mode in the top right. Choose "Load unpacked" and select the extracted "hq-dataprofiler-chrome-extension" folder.

3. Pin it!

Click the jigsaw piece icon in the top right and then the pin icon. Now, launch HQ Data Profiler from the top right!
